🛡️ Centinela

HTTP Security Scanner

¿Qué es Centinela?

Centinela es un escáner de seguridad diseñado para auditar aplicaciones web. Detecta vulnerabilidades comunes relacionadas con configuraciones HTTP, incluyendo:

  • Falta de cabeceras de seguridad (CSP, HSTS, X-Frame-Options).
  • Políticas de CORS excesivamente permisivas.
  • Cookies expuestas sin flags `Secure` o `HttpOnly`.
  • Rutas sensibles expuestas (`/.git/config`, `/.env`, `/admin`).

API Endpoints

Escanea una URL enviando un request a la API:

curl -X POST https://centinela-br0a.onrender.com/api/scan \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com"}'

Descarga un reporte PDF del escaneo:

curl -OJ https://centinela-br0a.onrender.com/api/scan/:id/report.pdf

CLI

También puedes usar Centinela desde la terminal:

npx centinela scan https://example.com --output reporte.pdf
